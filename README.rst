git-semver
==========

Install
:::::::

.. code-block:: shell

    (sudo) pip install git-semver

Usage
:::::

.. code-block:: shell

    $ git semver
    0.1.0

.. code-block:: shell

    $ git semver --next-patch
    0.1.1

.. code-block:: shell

    $ git semver --next-minor
    0.2.0

.. code-block:: shell

    $ git semver --next-major
    1.0.0
